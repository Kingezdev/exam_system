Program Information

What is a program?

An academic program is a structured course of study leading to a degree or certificate. Programs are typically organized by departments and include specific requirements for completion.

Program details:

  • Name: Full program name (e.g., Bachelor of Science in Computer Science)
  • Code: Short identifier (e.g., BSCS)
  • Department: Academic department offering the program
  • Duration: Length of study in years
  • Description: Program overview and objectives
Tip: Make sure the program code is unique and follows your institution's naming conventions.
Guidelines

Before creating a program:

  • Ensure the department exists
  • Verify program code uniqueness
  • Prepare program description
  • Confirm duration requirements
Important: Programs with existing enrollments cannot be easily modified. Ensure all details are correct before creation.
